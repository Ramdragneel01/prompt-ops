# Security Policy

## Supported Version

Latest main branch is supported.

## Reporting

1. Do not disclose sensitive issues in public issues.
2. Report vulnerabilities privately to maintainers.
3. Provide reproduction steps and impact details.

## Baseline Controls

1. Host allowlist enforcement.
2. Request payload size guard.
3. CORS allowlist for browser clients.
4. Optional HSTS header in production.
5. No committed secrets.
